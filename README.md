# A-climate-network-reveals-global-connections-among-major-tipping-elements

The Python codes for our paper "A-climate-network-reveals-global-connections-among-major-tipping-elements"

The fold named Identification of connections contains the codes for identifying robust global connections among tipping elements.

The fold named Connection strength analysis contains the codes for analyzing and plotting the connection strength among key global tipping elements.

The fold named Network structure analysis contains the codes for analyzing and plotting the centrality of key tipping elements within the climate network.

The fold named Path contains the codes for calculating and plotting the shortest paths between key tipping elements.
